Quick Start

1. Unzip the downloaded package into your project root.

2. Open index.html in your browser ‚Äî everything works out of the box.

3. To serve locally:

npx serve . # or with Python: python -m http.server 8080

4. To customize branding, open assets/css/style.css and edit the CSS variables under :root.

Colors & Design Tokens

All colors are defined as CSS custom properties inside :root (dark theme) and [data-theme="light"] (light theme overrides). This makes theming straightforward without touching component styles.

/* Core palette ‚Äî assets/css/style.css */ :root { /* Gold accent */ --gold-primary: #c49c5a; --gold-dim: #a07a3e; --gold-light: #d4b070; /* Background layers */ --bg-primary: #0e1016; /* page bg */ --bg-secondary: #13161e; /* sidebar bg */ --bg-elevated: #1a1d27; /* elevated elements */ --card-bg: #161a24; /* Status colors */ --success: #4ade80; --warning: #fbbf24; --danger: #f87171; --info: #60a5fa; }

Changing the Accent Color

Replace all three --gold-* variables to switch the accent color across the entire UI:

/* Example: switch to indigo accent */ :root { --gold-primary: #818cf8; --gold-dim: #6366f1; --gold-light: #a5b4fc; }

Dark / Light Mode

Theme state is controlled by the data-theme attribute on <html>. The toggle button in the header switches this attribute and saves the preference in localStorage.

/* Force light mode on load: */ <html data-theme="light"> /* Or initialize via JS: */ document.documentElement.setAttribute('data-theme', 'light');
‚ö†Ô∏è Note: chart colors are recalculated after theme changes. The getChartColors() helper in app.js reads CSS variables at runtime ‚Äî re-render charts on theme change if updating dynamically.

Charts (ApexCharts)

Charts are initialized in both inline <script> blocks per page and in assets/js/charts.js for the main overview. Always target an element ID and call .render().

// Minimal ApexCharts line chart new ApexCharts(document.getElementById('myChart'), { chart: { type: 'line', height: 220 }, series: [{ name: 'Sales', data: [30, 40, 35, 50, 49, 60] }], colors: ['#c49c5a'], stroke: { curve: 'smooth', width: 2 }, tooltip: { theme: 'dark' }, }).render();

Use window.VeloraApp?.getChartColors() to get theme-aware color tokens (grid color, text color, etc.) for consistent styling across themes.

Data Tables

Wrap any <table class="data-table"> with <div class="table-wrapper"> to enable horizontal scrolling on mobile. Enable pagination with:

// Enable pagination: tableId, rowsPerPage window.VeloraApp?.initPagination('myTable', 8);

Add a <div class="pagination" id="myTablePagination"></div> immediately after the table wrapper for pagination controls to render.
